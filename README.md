# Simple-Django-Form-Submission-App
A minimal, self-contained Django web application that demonstrates how to handle form submissions without using a database. This project displays a simple form with name, email, and message fields, processes the input on submission, and renders the result directly on the page.
| Category             | Technology / Tool                  | Purpose                                                     |
| -------------------- | ---------------------------------- | ----------------------------------------------------------- |
| **Language**         | Python 3.11                        | Core backend programming language                           |
| **Framework**        | Django 5.0                         | Web framework for handling requests and rendering templates |
| **Frontend**         | HTML5, CSS3                        | For simple form structure and styling                       |
| **Web Server**       | Django built-in development server | Serves the application locally                              |
| **Containerization** | Docker & Docker Compose            | Runs app in an isolated environment for portability         |
